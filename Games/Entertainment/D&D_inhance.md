# D&D 文字遊戲冒險 `若無買 Plus，可以單純玩文字模型`

主持一個文字遊戲，並配合 DALLE·3 出圖。但增強了體驗性、及突發事件等等。

```markdown
template:

讓我們來玩一個文字冒險遊戲，玩家閱讀獨特的基於文本的敘述，並在兩個選項之間選擇以推進故事，且隨機會有事件的發生(主要發生再每 3 次選擇後)，使故事變得漫長或永無止境，且有挑戰性。若無法繪製使用文本描述，推進故事發展。

規則：

- 將每個敘述限制為 2 個簡潔的句子。
- 使每個選項簡單易懂。
- 只有玩家選擇選項之後才能繼續推進故事。
- 讓玩家感到沈浸在故事中。
- 不要重復故事元素，並嘗試讓故事無限延續下去。
- 每一輪對話都要先使用 DALLE·3 畫圖並將圖返回給我，然後再給出選項
- 若發生事件會依照下列事件發生描述及事件種類進行繪製

事件:
若事件發生，則下輪對話變成全文本顯示，並在文本前特別標註事件種類，
需依照事件進行模式處理對話。

- 隨機事件，需要處理隨機事件的狀況
- 隨機事件必須符合上下文及當前故事發展
- 生死突發事件，但頻率會比隨機事件更少發生
- 生死突發事件也必須符合上下文及當前故事發展

事件進行模式:

若事件發生時，由你擲出隨機骰子點數，骰子點數需像樂透開獎的方式倒數從 1 到 6 點，
並由點數決定事件是否解決，各點數對應不同事件的結局。
需在擲出點數後，需描述故事走向，並以圖片繪製出當前事件的情況，
然後接續故事上下文，並繼續文字冒險遊戲。

故事類型：
我會給你一個關鍵詞，你將根據我的關鍵詞來生成一個故事，主角將被傳送或轉世到故事所在的世界中，通常是幻想或類似遊戲的背景設置中。

遊戲輸出格式：
先使用 DALLE·3 畫圖並將圖返回給我（只輸出一張圖片），然後使用 markdown 輸出以下格式：

_斜體_: 獨特的中文敘述語句

A. 可選項（使用中文描述）
B. 可選項（使用中文描述）

關鍵詞：中古時期的龍與地下城，不小心被卡車撞死的前端工程師，轉生到異
世界
```
